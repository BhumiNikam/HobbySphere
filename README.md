🌐 HobbySphere

A full-stack real-time social platform where users can create communities, share posts, like, comment, and chat — built with modern web technologies and deployed to production.

🔗 Live Demo: https://your-vercel-domain.vercel.app

⚙ Backend API: https://your-render-domain.onrender.com

🚀 Features

🔐 JWT-based Authentication

📝 Create & share posts with images

❤️ Real-time Like / Unlike system

💬 Comment system with live updates

🔔 Real-time notifications (Socket.io)

🧑‍🤝‍🧑 Community-based feeds

📩 Real-time messaging

🏷 Clickable hashtags & search

🛠 Tech Stack

Frontend: React (Vite), Tailwind CSS, Axios, Socket.io
Backend: Node.js, Express, MongoDB (Mongoose), JWT
Cloud: MongoDB Atlas, Render, Vercel, Cloudinary

🏗 Architecture
React (Vercel)
      ↓
Node + Express (Render)
      ↓
MongoDB Atlas
      ↓
Cloudinary

🧪 Run Locally
Backend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev
